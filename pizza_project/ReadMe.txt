17. Чат-поддержка

Веб-приложение DJANGO

*** Админка вынесена в спец url с правами доступа только админ

Только админ может добавить новую пиццу

Админу видны следующие данные:  количество сообщений в техподдержку, количество заказов по статусу, выручка и количество выполненных заказов за день и за месяц

Страница техподдержки - видит все диалоги, может зайти и посмотреть историю каждого + ответить

Админ может зайти и посмотреть все заказы по нужному статусу. Затем зайти в каждый заказ,  поменять сам заказ и его статус


*** Телеграм-бот

Приветствие

Показать категории пицц

Текстом показать все пиццы по данной категории (состав + цена)

Кнопка вернуться назад

Если это возможно - добавить товар в корзину

Если это возможно - изменить состав товара

Вывести общий заказ (корзину) с кнопкой заказать

Спросить данные имя, адрес или самовывоз, телефон

Отправить уведомления о смене статуса заказа

Возможность написать в техподдержку 

После выдачи заказа пожелать приятного аппетита
